# Applications & Use Cases

## Pull-Based Contextual Services

### Navigation Example
1. User requests city navigation through [AIA agent](architecture/aia_agents.md)
2. Contributing clients share encrypted location/destination data
3. [Genome](architecture/genome.md) processes aggregated traffic patterns via MPC
4. Optimized routes returned with privacy preservation
5. Contributors rewarded based on data utility
