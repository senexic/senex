# Experimental coordination contracts — archived

:::danger No live contract or token
SENEXIC does not currently operate a public AIA Chain contract, live token, or
public reward network. This route is retained only for historical continuity.
:::

An early prototype used test-network contracts to explore coordination,
contribution records, and experimental accounting. That work was an
investigation, not a production protocol.

The previous public page included prototype interfaces, data structures,
parameters, and fee logic. Those details have been retired because they were
specific to an obsolete experiment, did not describe the current product, and
could be mistaken for a supported financial or security design.

The current SENEXIC direction separates:

- local AIA foundations that can be evaluated today;
- V1-testnet targets that still require qualification;
- research into future verification and settlement; and
- any future economic design, which remains subject to independent technical
  and legal review.

Read the [Intelligent Chain direction](../../architecture/intelligent_chain.md)
or the [contribution-economy research overview](../../tokenomics/index.md).
