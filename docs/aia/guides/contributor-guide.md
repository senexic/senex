# Legacy AIA Chain contributor guide

:::caution Retired instructions
The earlier contributor workflow is no longer active. Do not use old wallet,
test-network, fee, key-management, or validator instructions as guidance for
the current AIA product.
:::

The prototype contributor flow was created to test a narrow research question:
could independent participants request and verify AI work in a shared
experimental environment? It was not a production service and did not establish
a live contribution economy.

Current AIA work is local-first and owner-centered. Participation is intended to
be explicit, permission-bounded, and revocable. Future V1-testnet participation
will require its own reviewed onboarding, safety guidance, and test-only value
policy before public instructions are published.

Until that qualification is complete:

- there is no public validator onboarding;
- there is no live contribution payment;
- there is no transferable ASHA asset;
- no legacy configuration should be treated as supported; and
- any future participation guide will be versioned against a named release.

Continue with the [current AIA overview](../../architecture/aia_agents.md) and
[roadmap](../../roadmap.md).
